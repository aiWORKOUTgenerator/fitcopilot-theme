# ButtonGroup

The `ButtonGroup` component organizes multiple buttons with consistent spacing and alignment.
It helps create visually cohesive button layouts for forms, dialogs, and action areas.

## Features

- **Horizontal or vertical** layout options
- **Configurable spacing** between buttons
- **Alignment control** for button positioning
- **Equal width option** for uniform button sizes
- **Theme variant support** across all FitCopilot themes

## Usage

ButtonGroup provides control over button organization while preserving the theme system.

```tsx
import { Button, ButtonGroup } from '../features/shared/Button';

// Basic button group
<ButtonGroup>
  <Button variant="secondary">Cancel</Button>
  <Button variant="primary">Submit</Button>
</ButtonGroup>

// Vertical button group
<ButtonGroup direction="vertical">
  <Button variant="primary">Main Action</Button>
  <Button variant="secondary">Secondary Action</Button>
  <Button variant="text">Text Action</Button>
</ButtonGroup>

// Right-aligned button group
<ButtonGroup alignment="end">
  <Button variant="secondary">Cancel</Button>
  <Button variant="primary">Submit</Button>
</ButtonGroup>

// Equal width buttons
<ButtonGroup equalWidth>
  <Button variant="secondary">No</Button>
  <Button variant="primary">Yes</Button>
</ButtonGroup>
```

## Theme Variants

ButtonGroup automatically inherits the active theme and applies the appropriate CSS variables
for spacing based on the current theme context.

## Accessibility

- ButtonGroup uses `role="group"` to indicate related buttons
- Always provide an `aria-label` for button groups without visual labels
- Maintain sufficient spacing for touch targets

## Implementation Details

ButtonGroup uses CSS variables and data attributes for theme-specific styling:

```scss
// Theme variants
[data-theme="default"] & {
  --button-group-spacing: var(--spacing-sm, 0.5rem);
}

[data-theme="gym"] & {
  --button-group-spacing: var(--spacing-md, 0.75rem);
}

[data-theme="sports"] & {
  --button-group-spacing: var(--spacing-sm, 0.5rem);
}

[data-theme="wellness"] & {
  --button-group-spacing: var(--spacing-xs, 0.25rem);
}
```
