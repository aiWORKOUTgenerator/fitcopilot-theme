## Typography

FitCopilot uses a type system built on [Inter](https://fonts.google.com/specimen/Inter) for UI elements and [Merriweather](https://fonts.google.com/specimen/Merriweather) for headlines. This combination provides excellent readability across devices while maintaining our brand personality.

### Font Stacks

```css
--font-family-heading: 'Merriweather', Georgia, serif;
--font-family-body: 'Inter', system-ui, -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 'Helvetica Neue', Arial, sans-serif;
```

### Font Sizes

Our typographic scale follows an exponential ratio of 1.25 (major third) to maintain visual harmony.

```css
--font-size-xs: 0.75rem;   /* 12px */
--font-size-sm: 0.875rem;  /* 14px */
--font-size-base: 1rem;    /* 16px */
--font-size-lg: 1.125rem;  /* 18px */
--font-size-xl: 1.25rem;   /* 20px */
--font-size-2xl: 1.5rem;   /* 24px */
--font-size-3xl: 1.875rem; /* 30px */
--font-size-4xl: 2.25rem;  /* 36px */
--font-size-5xl: 3rem;     /* 48px */
--font-size-6xl: 3.75rem;  /* 60px */
```

### Line Heights

Proper line heights ensure readability and aesthetic balance.

```css
--line-height-tight: 1.25;
--line-height-snug: 1.375;
--line-height-normal: 1.5;
--line-height-relaxed: 1.625;
--line-height-loose: 2;
```

## Border Radius

Consistent border radius values maintain visual cohesion.

```css

--radius-lg: 0.5rem;    /* 8px */
```

## Shadows

Shadows provide elevation cues and visual hierarchy.

```css
--shadow-sm: 0 1px 2px 0 rgb(0 0 0 / 0.05);
--shadow-md: 0 4px 6px -1px rgb(0 0 0 / 0.1), 0 2px 4px -2px rgb(0 0 0 / 0.1);
--shadow-lg: 0 10px 15px -3px rgb(0 0 0 / 0.1), 0 4px 6px -4px rgb(0 0 0 / 0.1);
--shadow-xl: 0 20px 25px -5px rgb(0 0 0 / 0.1), 0 8px 10px -6px rgb(0 0 0 / 0.1);
--shadow-2xl: 0 25px 50px -12px rgb(0 0 0 / 0.25);
--shadow-inner: inset 0 2px 4px 0 rgb(0 0 0 / 0.05);
```

## Implementation Guide

These tokens should be used consistently throughout the application, both in component styles and in any custom CSS.

### CSS Implementation

```css
:root {
  /* Colors */
  --color-primary-500: #0ea5e9;
  --color-secondary-500: #ec4899;
  
  /* Typography */
  --font-family-heading: 'Merriweather', Georgia, serif;
  --font-size-base: 1rem;
  
  /* Spacing */
  --space-4: 1rem;
  
  /* Other tokens... */
}

.button {
  background-color: var(--color-primary-500);
  padding: var(--space-2) var(--space-4);
  border-radius: var(--radius-md);
  font-family: var(--font-family-body);
}
```

### React & Tailwind Implementation

Our system leverages Tailwind CSS for consistent styling:

```jsx
function PrimaryButton({ children }) {
  return (
    <button className="bg-primary-500 hover:bg-primary-600 text-white px-4 py-2 rounded-md">
      {children}
    </button>
  );
}
```
