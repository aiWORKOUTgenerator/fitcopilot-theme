FitCopilot uses a carefully selected typography system to ensure readability and brand consistency across all platforms.

## Font Family

For code examples, we use **Fira Code**, a monospace font with programming ligatures.

## Usage Guidelines

- Maintain consistent heading hierarchy (h1 → h2 → h3 → h4) for proper document structure
- Use appropriate font weights to create visual hierarchy
- Avoid using too many font sizes on the same page
- Ensure sufficient contrast between text and background colors
- Maintain a line height of 1.5 for body text to ensure readability
- Keep line length between 60-80 characters for optimal readability
